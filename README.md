## 📋 О проекте

> 🎓 **Учебный проект** | 📅 **Производственная практика** | 🏫 **III курс**

Данный проект разработан в рамках **производственной практики** студентами **III курса** по специальности **09.02.07 «Информационные системы и программирование»**.

Проект представляет собой единый информационный веб-ресурс для двух дачных поселков:

| | | |
|:---:|:---:|:---:|
| 🌳 | **ДПК «Заповедное»** | Уютный дачный поселок в экологически чистом районе |
| 🌾 | **ДПК «Колосок»** | Комфортный дачный поселок с развитой инфраструктурой |
